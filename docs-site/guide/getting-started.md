# 开始使用

## 安装

```bash
npm i sculp-js
```

## 按需导入（推荐）

```ts
import { cloneDeep } from 'sculp-js';
```

如果你希望更细粒度按模块导入（配合打包器 tree-shaking），也可以：

```ts
import { cloneDeep } from 'sculp-js/cloneDeep';
```

## 浏览器 CDN

```html
<script src="https://unpkg.com/sculp-js"></script>
```
